# **App Name**: SkillShowcase

## Core Features:

- Talent Profile Creation: Enable talents to create detailed profiles showcasing their skills, experience, and portfolio items.
- Skill Search and Filtering: Implement a robust search engine allowing recruiters to find talents based on skills, location, availability, and experience level.
- Portfolio Showcase: Allow talents to upload and display their work samples, including images, PDFs, videos, and external links.
- Direct Contact: Enable recruiters to directly contact talents via email or an internal messaging system. Allow user to specify WhatsApp if desired
- Skill Tag Suggestion: Suggest relevant skill tags to users based on their profile information and portfolio content, enhancing discoverability. LLM serves as a tool, improving suggested tags by assessing its relevance and importance.

## Style Guidelines:

- Primary color: A warm coral (#FF7F50) to evoke creativity and connection.
- Background color: Light beige (#F5F5DC) provides a neutral and clean backdrop.
- Accent color: Sky blue (#87CEEB) to highlight calls to action and important information.
- Body text: 'PT Sans', a humanist sans-serif font to combine a modern look and a little warmth or personality; Suitable for headlines or body text
- Card-based layouts to display talent profiles and search results in a visually appealing and organized manner.
- Use clear and intuitive icons to represent different skill categories and functionalities.
- Subtle transitions and animations to enhance user engagement and provide visual feedback during interactions.
